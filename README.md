# 📊 Power BI Sales Dashboard

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** designed to analyze and visualize sales performance.
The dashboard provides insights into sales trends, regional performance, product categories, and key business metrics.

The goal of this project is to demonstrate **data visualization, data modeling, and business insight generation using Power BI**.

---

## 🛠 Tools & Technologies Used

* Power BI
* Excel / CSV Dataset
* Power Query
* DAX (Data Analysis Expressions)

---

## 📂 Project Files

| File                 | Description                  |
| -------------------- | ---------------------------- |
| `power bi data.pbix` | Main Power BI dashboard file |
| `Data_set 2.csv`     | Dataset used for analysis    |
